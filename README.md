# ABP 2do Quimestre - Control de Inventario (Tienda de Ropa Coral)
 
## 1. Contexto del problema
Actualmente, la tienda de ropa "Coral" lleva el control del inventario de manera manual, utilizando cuadernos o notas, lo que provoca pérdida de información, errores en el conteo de productos, duplicidad de registros y dificultad para conocer el stock real disponible.  
Esto genera problemas al momento de reponer productos y atender correctamente a los clientes.
 
Por esta razón, se plantea desarrollar una aplicación que permita registrar, consultar y actualizar el inventario de forma digital, segura y ordenada.
 
## 2. Análisis de requerimientos
La aplicación permitirá:
- Registrar prendas de vestir (nombre, marca, precio, tipo de tela, país de fabricación, stock).
- Consultar el inventario completo.
- Buscar una prenda por nombre o marca.
- Actualizar el stock de una prenda.
- Eliminar prendas del inventario.
- Validar que no se ingresen datos vacíos o incorrectos.
